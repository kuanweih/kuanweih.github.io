## 🛠️ Setup Instructions

### Option 1: Quick Deploy to GitHub Pages

1. **Create a new repository** named `yourusername.github.io` (replace `yourusername` with your GitHub username)

2. **Clone the repository**:

   ```bash
   git clone https://github.com/yourusername/yourusername.github.io.git
   cd yourusername.github.io
   ```

3. **Create the folder structure**:

   ```bash
   mkdir css js assets assets/images
   ```

4. **Add the files**:

   - Copy the HTML content to `index.html`
   - Copy the CSS content to `css/styles.css`
   - Copy the JavaScript content to `js/script.js`
   - Copy this README to `README.md`

5. **Commit and push**:

   ```bash
   git add .
   git commit -m "Initial website setup"
   git push origin main
   ```

6. **Enable GitHub Pages**:
   - Go to your repository settings
   - Scroll to "Pages" section
   - Select "Deploy from a branch"
   - Choose "main" branch and "/ (root)" folder
   - Save

Your website will be live at `https://yourusername.github.io` within a few minutes!

### Option 2: Use Existing Repository

If you want to use a different repository name:

1. Create a new repository with any name
2. Follow steps 2-5 above
3. In repository settings → Pages, enable GitHub Pages
4. Your site will be at `https://yourusername.github.io/repository-name`

## ✏️ Customization Guide

### 1. Personal Information

Edit `index.html` and replace:

- `"Your Name"` with your actual name
- `yourname@university.edu` with your email
- All placeholder content with your actual information
- Research projects, publications, and credentials

### 2. Styling Customization

Edit `css/styles.css` to modify:

- **Colors**: Change the `#8b7355` color scheme throughout
- **Fonts**: Modify the Google Fonts import and font-family declarations
- **Layout**: Adjust grid layouts, spacing, and responsive breakpoints
- **Animations**: Customize timing and effects

### 3. Content Sections

The website includes these main sections:

- **Hero**: Landing section with name and tagline
- **About**: Personal background and credentials
- **Research**: Current projects and research areas
- **Publications**: Academic publications list
- **Contact**: Contact information and social links

### 4. Adding New Features

To add new sections:

1. Add HTML structure in `index.html`
2. Add corresponding styles in `css/styles.css`
3. Add any interactive features in `js/script.js`
4. Update navigation links

## 🎨 Design Philosophy

This website follows several design principles:

- **Minimalism**: Clean, uncluttered layout focusing on content
- **Academic Elegance**: Professional appearance suitable for academic contexts
- **Cultural Touch**: Subtle Chinese calligraphy elements adding personality
- **Performance**: Optimized for fast loading and smooth interactions

## 📱 Responsive Features

The website adapts to different screen sizes:
